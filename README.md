# Tongue 🐸

A physics tongue-swing game. Grapple, swing, and leap a frog toward the goal.

Built with **Godot 4.7**. Currently a greybox prototype.

## Controls

| Input | Action |
|-------|--------|
| Mouse (hold left) | Shoot / reel tongue |
| A / D | Move, air-steer |
| Space | Jump |
| R | Reset |
| F | Free-cam toggle |

## Run it

Open the project in Godot 4.7 and press Play (`Main.tscn`).

## Layout

- `Game.gd` — gameplay (tuning constants up top)
- `Main.tscn` — main scene
- `assets/` — sprites
